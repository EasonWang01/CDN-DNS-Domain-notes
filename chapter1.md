HTML Tags
------------------
1.基本構造--原始人
```
<!DOCTYPE html>
<html>
<head>
<!--我是注解--> 
</head>
<body>
<!--我是注解--> 
</body>
</html>
```
2.加上CSS--原始人穿上衣服
```
<style> 

body{
 background:black;
} 

</style>
```
3.加上javascript--原始人發現了電
```
<script> 
alert('Hello,World') 
</script>
```
